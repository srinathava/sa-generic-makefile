Example of project with subdirectories
======================================
This test directory is laid out with the following structure:

  PWD/
    src/
      src/foo/
        src/foo/foo.cpp
        src/foo/foo2.cpp
        src/foo/foo.hpp
      src/bar/
        src/bar/bar.cpp

The executable and objects are placed in bin/ and obj/ subdirectories of
this directory.
